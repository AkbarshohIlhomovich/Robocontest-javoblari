# M074D — Xor va summa

**Vaqt limiti**: 1000 ms  
**Xotira limiti**: 32 MB  
**Qiyinchilik**: 1%  
**Kategoriya**: 

**Manba**: https://robocontest.uz/tasks/M074D

## Masala matni

Sizga L soni berilgan. Quyidagi shartlarni qanoatlantiradigan juftliklar sonini hisoblang:
\(a + b \le L\)
\(a + b = a\ XOR\ b\)
Kiruvchi ma'lumotlar:
Kirish faylida L sonining binar ko'rinishi beriladi.
\(1 \le L < 2^{100\ 001}\)
Chiquvchi ma'lumotlar:
Javob juda katta bo'lishi mumkin, shuning uchun javobni
\(10^9+7\)
ga bo'lgandagi qoldiqni chop eting.
