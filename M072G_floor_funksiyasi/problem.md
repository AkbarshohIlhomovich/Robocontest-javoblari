# M072G — Floor funksiyasi

**Vaqt limiti**: 1000 ms  
**Xotira limiti**: 32 MB  
**Qiyinchilik**: 1%  
**Kategoriya**: 

**Manba**: https://robocontest.uz/tasks/M072G

## Masala matni

A, B va N sonlari berilgan.
\(floor(Ax/B) - A*floor(x/B)\)
ning maksimum mumkin bo'lgan qiymatini aniqlang. Bu yerda
\(x\)
manfiy bo'lmagan va N dan katta bo'lmagan butun son.
Bu yerda
\(floor(t)\)
\(t\)
dan katta bo'lmagan eng katta butun sonni bildiradi.
Kiruvchi ma'lumotlar:
Kirish faylida A, B, N sonlari kiritiladi.
\(1 \le A \le 10^6\)
\(1 \le B, N \le 10^{12}\)
Chiquvchi ma'lumotlar:
Maksimum mumkin bo'lgan qiymatni chop eting.
